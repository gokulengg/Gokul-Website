# Sri Gokul Engineering Works Website

A modern, fast website built with Vite, React, TypeScript, Tailwind CSS, and shadcn-ui.

## Tech Stack

- Vite (React + SWC)
- TypeScript
- Tailwind CSS
- shadcn-ui (Radix UI + utilities)

## Local Development

Prerequisites: Node.js 18+ and npm.

```sh
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

- `src/pages` – App pages (Home, Projects, Services, Facilities, About, Contact)
- `src/components` – Reusable UI components
- `src/assets` – Local assets used by the app
- `public` – Static files served at the site root (images, videos, robots.txt)

## Deployment

This project works great on Vercel, Netlify, or any static hosting that supports SPA routing.

On Vercel, set the framework preset to Vite and ensure the Output Directory is `dist`.
